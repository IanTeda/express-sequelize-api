About The Project

This is a repository template for starting a RESTful API server, with basic user functionally already added.

Built With

  • Sequelize
  • Passport

Getting Started

To get a local copy up and running follow these simple steps.

Prerequisites

This is an example of how to list things you need to use the software and how to install them.

  • npm
npm install npm@latest -g

Installation

  1. Clone the repo
git clone https://github.com/ianteda/express-sequelize-api.git
  1. Install NPM packages
npm install
  1. Create and edit your local .ENV file
cp env.example .env
code .env
  1. Run Express server
npm start
